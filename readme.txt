you need node.js installed

in the root folder sample:
	$ npm install
	$ npm start

	Experiencia

	A un principio requiere dos recargas para guardar los datos pero una vez con los datos en la base de datos local la aplicación se obtiene un comportamiento esperado aún sin el acceso a internet.

	Complicaciones

	No mayor complicación que la de identificar los lugares donde se utilizaba IndexDB e implementar su contraparte en Dexie.js.
